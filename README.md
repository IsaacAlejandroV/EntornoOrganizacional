# EntornoOrganizacional

NonaSoft es una startup tecnológica mexicana de 100 colaboradores, especializada en desarrollo de software a la medida y soluciones SaaS para los sectores financiero y retail. Opera con equipos distribuidos en Ciudad de México, Monterrey y Guadalajara, con toda su infraestructura sobre Ubuntu Server 22.04 y Docker.


## Áreas Tecnológicas y Herramientas Seleccionadas

| Área | Herramienta Seleccionada | Licencia |
|------|--------------------------|----------|
| Comunicación interna | **Mattermost CE** | MIT |
| Gestión de código y proyectos | **GitLab CE** | MIT |
| Gestión documental | **Nextcloud Hub + Collabora** | AGPL v3 |
| CI/CD e infraestructura | **Woodpecker CI** | Apache 2.0 |
