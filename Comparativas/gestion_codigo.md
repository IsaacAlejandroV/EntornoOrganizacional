**Área**: Gestión de código fuente y proyectos de desarrollo  
**Organización**: NovaSoft Solutions

### 1. GitLab Community Edition

| **Licencia** | MIT |
| **Último commit** | Abril 2025 |
| **Contribuidores** | ~3,000 en GitHub |
| **Documentación** | Excelente — https://docs.gitlab.com |
| **Curva de aprendizaje** | Media |
| **Compatibilidad** | Docker, Kubernetes, Linux, S3, LDAP/SSO |

**Pros**: CI/CD nativo integrado, Container Registry, issues, wikis, tableros Kanban, MR reviews. Todo en una sola plataforma.  
**Contras**: Requiere mínimo 4 GB RAM, instalación más pesada que alternativas ligeras.

### 2. Gitea

| **Licencia** | MIT |
| **Último commit** | Abril 2025 |
| **Contribuidores** | ~500 en GitHub |
| **Documentación** | Buena — https://docs.gitea.com |
| **Curva de aprendizaje** | Baja |
| **Compatibilidad** | Linux, Docker, muy ligero (~150 MB RAM) |

**Pros**: Extremadamente ligero, fácil de instalar, interfaz muy similar a GitHub.  
**Contras**: No incluye CI/CD nativo completo, menor ecosistema de integraciones.

### 3. Forgejo

| **Licencia** | GPL v3 |
| **Último commit** | Abril 2025 |
| **Contribuidores** | ~200 en GitHub |
| **Documentación** | Buena — https://forgejo.org/docs |
| **Curva de aprendizaje** | Baja |
| **Compatibilidad** | Linux, Docker, fork compatible con Gitea |

**Pros**: Fork comunitario de Gitea con gobernanza más abierta, desarrollo activo.  
**Contras**: Comunidad más pequeña, menor ecosistema que GitLab.

##  Herramienta Seleccionada: **GitLab CE**

**Justificación**: GitLab CE es la única opción que integra en una sola plataforma todo lo que NovaSoft necesita: control de versiones, CI/CD, gestión de issues y Container Registry. Elimina la necesidad de múltiples herramientas y reduce la complejidad operativa.
